## Como testar no piloto
Casos mínimos:
1. Orçamento residencial completo
2. Pintura
3. Piso
4. Parede/alvenaria
5. Telhado/cobertura
6. Hidráulica/esgoto
7. Elétrica
8. Patologia
9. Relato técnico
10. Geometria simples

## Critério de aprovação do piloto
- Usuário entende a resposta
- Usuário consegue copiar e colar o orçamento
- Usuário percebe as premissas
- Usuário sabe o que está pendente
- Usuário não confunde estimativa preliminar com proposta final
